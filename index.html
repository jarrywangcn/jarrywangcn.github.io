移民智利

09 Dec 2016, 23:38
dream
chile / visa / passport

1. 为什么想移民 作为一个喜欢旅行的人来说，想移民倒不是为了能更方便的去更多的国家玩，只要有了工作证明和存款证明，中国护照申请签证并没有那么麻烦。 一四年我和马修分手，那时候我连港澳通行证都没有，不然就直接飞香港了。 一四年十一月他在美国联系我，要不是没有美国签证，就算那时候穷，机票再贵我借钱都过去见他。他回香港，那时候我办了港澳，还签了两次香港的签注，买好机票，他跟我说自己回美国了。 今年六月在台湾，联系上的时候他在重庆，在台北的那晚我都下定决心提早结束台湾的旅行飞重庆见他，问他在哪，他给我发了个定位在西澳大利亚，我真他妈的… 事情可能也没我说的这么简单，如果我真的是为了马修想要有随时可以走的护照，我为什么不直接移民香港呢？但我不想深思这个问题 2. 为什么想移民智利 2.1 智利移民门槛低，流程简便 门槛低：50万人民币存款或者每个月被动收入有9000人民币左右，移民费用在10万人民币以内 流程简便：第一年在智利本土待够183天，就可以拿到绿卡（永久居留权），之后四年每年在智利待一天，就可申请入籍，拿到智利护照 2.2 智利护照免签国众多 截止2016年2月，全球共有155个国家对持智利护照的人士施行免签或者落地签政策 申根地区：单个国家90d，总计180d 香港：3m 阿根廷：90d 加拿大：6m 以色列：3m 日本：90d 墨西哥：180d 摩洛哥：3m 新西兰：90d 俄罗斯：90d 泰国：3m 土耳其：3m 英国：6m 美国：90d（ESTA 2y） 台湾：90d 澳门：30d 真的可以随意在世界各地更换居住地～超过3个月换一个地方住～ 3. 定个小目标 二零一六年十二月定个小目标 二零一八年拿到智利绿卡 参考链接 性价比最高的发达国家移民项目——智利 Visa requirements for Chilean citizens

使用七牛云+dropzone在静态博客中插入图片

08 Dec 2016, 20:21
programming
qiniu / dropzone

用wordpress, medium之类的博客服务，想要在文章中插入图片，网站有自己的服务器可以上传图片。如果自己架设网站，就需要把照片上传到互联网，获取网址，才能利用markdown语法在文章中插入图片。 可以选择例如flickr这样的网络相册（会提供照片外链网址），考虑到flickr已经被墙了，遂放弃，也不想为此重新注册一个网络相册。 也可以选择例如七牛、github，把图片上传到服务器，获得外链。本来想选github后来听说官方不支持这种用法，外加看到一篇使用七牛云+dropzone拖拽上传图片自动复制外链到剪贴板的文章，就决定使用七牛云。 操作步骤 注册七牛云，并创建存储空间 在appstore上购买dropzone 3 (12人民币)，再去官网下载沙箱版dropzone3 在untested action中选择qiniu插件install 配置插件，操作见使用Dropzone和七牛优化博客图床 尝试拖拽上传文件，生成外链，使用⬇️(在第四步中选择了本地图片备份，在选择的目录下也出现了上传的文件) 压缩图片之后上传 目前使用的是苹果的预览工具，想要在dropzone里实现… 苹果预览修改尺寸效果如下 update 使用dropzone插件1024px，但感觉1024像素在博客中还是有些大，自己修改了代码调成800，缺陷是自动覆盖原文件，所以最好是先备份再压缩，但已经挺方便了，满足。

build personal blog

08 Dec 2016, 17:16
programming
programming / hugo / github

之前尝试过很多次搭建个人博客，看到Wilson Lee的网站使用format做portfolio，效果很棒，但收费挺高。买过几个域名，挺久之前买过jarry.wang 到期了也没用过一次，七月买了jarrywang.de。花了挺大功夫，自建博客很多人推荐用wordpress，但我觉得wordpress相当麻烦，很厚重，用了个其他的（忘记叫啥）。当时托管在亚马逊aws，创建instance解析ip地址又弄了很久，但还是搞定了。就是在网址上输入xx.xx.xx，能看到网页。死在了最后一步，把解析的ip地址和域名绑定，forward只能跳转，还是会显示ip地址。弄了三天左右，最后放弃了… 今天下午弄了四个小时，就全部搞定了。方法是用hugo搭建网站，托管在github pages上，github会自动生成一个可访问的网站 your_username.github.io，在域名管理中加入三个DNS设置，要在project中加一个文件。 现在每次更新是先创建一个markdown文件，run hugo生成public文件夹，用git推送到在github托管的项目（目前用的是source tree，一款gui方式使用git的软件）。 总结： 今天弄的这么顺利和这两个月学会的命令行、git、github起到了很大的作用。如果是之前只会用gui的话，可能真的要被wordpress弄死！dependency很重要！ 这两个月也习惯了看文档学东西，今天除了有些用户写的中文教程还看hugo的文档，碰到bug学会差stackoverflow，还去hugo的community里问了… 感天动地！ 参考链接： 使用hugo搭建个人博客站点 github怎么绑定自己的域名
